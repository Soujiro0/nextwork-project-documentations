## 📌 About This Showcase

Welcome to my **NextWork Projects Repository**! This repository serves as a centralized, structured knowledge base and showcase gallery for all my hands-on technical labs and projects completed through NextWork.

Every project in this repository includes:
- 📄 **Lesson PDF**: Exported high-resolution documentation of the entire walkthough.
- 📑 **Markdown Documentation**: A clean, GitHub-rendered architectural summary and step-by-step guide.
- 🖼️ **Visual Assets**: Diagrams and architectural charts used in the lesson.
- 💻 **Sample Code**: Configuration scripts, templates, or source code files (where applicable).
- 🌐 **Live Project URL**: Direct links to live deployments or cloud applications.

## ➕ How to Add Future Projects

This repository is architected for seamless scalability. Whenever you complete a new NextWork project:

1. Follow the step-by-step onboarding guide in [**ADD_PROJECT.md**](./ADD_PROJECT.md).
2. Copy the standardized template: `cp -r projects/_template projects/02-your-project-name`.
3. Drop in your PDF, update the project `README.md`, place images in `images/`, and paste your live link in `project-link.txt`.
4. Add a new row to the **Projects Showcase Gallery** table above!
