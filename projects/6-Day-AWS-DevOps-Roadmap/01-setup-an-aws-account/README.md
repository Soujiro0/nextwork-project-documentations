## Introducing Today's Project!

In this project, I'm going to set up my own AWS account, learn about the AWS Free Tier, and configure billing alerts to avoid surprise charges.

### Key tools and concepts

The key tools I used include user credentials, setting up billing alerts, and MFA to secure my account.  Key concepts I learnt include setup billing account to prevent surprise charges and adding MFA to secure my account and to prevent unauthorize access.

### Challenges and wins

This project took me approximately 20 minutes. The most challenging part was adding MFA to secure the account.

## Creating My AWS Account

In this step, I'm going to create my own new AWS account, then verify my email address, and then create a strong root password.

### Understanding the root user

I learned that the root user is the main account of my AWS account. It must have a strong password to prevent unauthorized access.

In this step, I'm going to set up my billing account, from selecting my free tier to adding a payment method to verify my account.

### Why AWS requires a payment method

I learned that AWS requires a payment method to verify the account identity.
